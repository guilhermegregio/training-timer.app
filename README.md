# Training Timer

**https://training-timer.app**

A free, open source, and ad-free PWA training timer for athletes and fitness enthusiasts. Built for speed, works offline, and installs on any device.

## Features

- **Stopwatch** - Count up with laps
- **Countdown** - Simple countdown timer
- **Intervals (HIIT)** - Work/rest rounds with customizable durations
- **EMOM** - Every Minute On the Minute
- **AMRAP** - As Many Rounds As Possible
- **For Time** - Race the clock with time cap
- **Custom Workouts** - Build complex workouts with text commands
- **Metronome** - Built-in BPM metronome for pacing
- **Workout Library** - Save, organize, and share your workouts
- **Voice Announcements** - Speak phase names and countdowns
- **Offline Support** - Full PWA, works without internet
- **Installable** - Add to home screen on any device
- **Share Workouts** - Share via URL with anyone

## Tech Stack

- **TypeScript** + **Vite** - Fast builds and type safety
- **PWA** - Service worker with offline caching
- **Web Audio API** - Low-latency sound effects and metronome
- **Cloudflare Pages** - Global edge deployment

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm run dev

# Run tests
pnpm test

# Build for production
pnpm run build

# Lint & format
pnpm run check
```
